Python Lists: FAQ

A list is a built-in data type that stores an ordered collection of items. Lists are mutable (changeable) and can contain different data types simultaneously.

You can create an empty list using square brackets my_list = [] or the constructor my_list = list().

append() adds a single element to the end of the list. extend() takes an iterable (like another list) and adds each of its elements to the list individually.

Python supports negative indexing. You can access the last element using my_list[-1].

Use the remove() method, e.g., my_list.remove('value'). Note: This only removes the first occurrence of that value.

pop() removes and returns an item at a given index (default is the last item). It is useful when you need to use the value after removing it.

Slicing uses the syntax list[start:stop:step]. For example, my_list[0:3] returns the first three elements.

You can use my_list.sort() to sort the list in place, or sorted(my_list) to return a new sorted list without modifying the original.
