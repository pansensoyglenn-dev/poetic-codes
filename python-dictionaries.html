Introduction to Python Dictionaries

In Python, dictionaries are unordered collections of key-value pairs. They are also known as associative arrays or hash maps in other programming languages.

Dictionaries are mutable, unordered, and indexed by keys (which must be immutable types like strings, numbers, or tuples). They are extremely efficient for lookups and are one of Python's most powerful data structures.

Key Characteristics:
  • Unordered collection of key-value pairs
  • Keys must be unique and immutable (strings, numbers, tuples)
  • Values can be of any data type
  • Mutable - can be changed after creation
  • Extremely fast for lookups (O(1) average time complexity)
# Basic dictionary example
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York",
    "is_student": False
}

print(person)
print("Type:", type(person))
print("Length:", len(person))
Output:
{'name': 'Alice', 'age': 30, 'city': 'New York', 'is_student': False}
Type: <class 'dict'>
Length: 4

Creating Dictionaries in Python

Python provides several ways to create dictionaries, each suitable for different scenarios.

Dictionary Creation Methods

Method Example Description
Curly braces {} {'a': 1, 'b': 2} Most common way, with key-value pairs
dict() constructor dict(a=1, b=2) Using keyword arguments (keys as identifiers)
From list of tuples dict([('a',1),('b',2)]) Convert list of (key, value) tuples
Dictionary comprehension {x: x*2 for x in range(5)} Create dynamically from iterable
fromkeys() method dict.fromkeys(['a','b'], 0) All keys with same initial value
# Different ways to create dictionaries
# 1. Using curly braces
dict1 = {"name": "John", "age": 25}

# 2. Using dict() constructor
dict2 = dict(name="Jane", age=30, city="Boston")

# 3. From list of tuples
dict3 = dict([('a', 1), ('b', 2), ('c', 3)])

# 4. Empty dictionary
empty_dict = {}
empty_dict2 = dict()

# 5. Nested dictionary
nested_dict = {
    "user1": {"name": "Alice", "score": 95},
    "user2": {"name": "Bob", "score": 87}
}

print("dict1:", dict1)
print("dict2:", dict2)
print("dict3:", dict3)
print("nested_dict:", nested_dict)

Accessing Dictionary Elements

There are several ways to access values in a dictionary, each with different behavior when keys don't exist.

# Accessing dictionary elements
student = {
    "name": "Emma",
    "age": 22,
    "courses": ["Math", "Physics", "Chemistry"],
    "grades": {"Math": 90, "Physics": 85, "Chemistry": 88}
}

# 1. Using square brackets (raises KeyError if key doesn't exist)
print("Name:", student["name"])
print("Courses:", student["courses"])
print("Math grade:", student["grades"]["Math"])

# 2. Using get() method (returns None or default if key doesn't exist)
print("Age:", student.get("age"))
print("Major:", student.get("major")) # Returns None
print("Major with default:", student.get("major", "Undeclared"))

# 3. Checking if key exists
print("'age' in student:", "age" in student)
print("'graduation_year' in student:", "graduation_year" in student)

# 4. Accessing all keys, values, and items
print("Keys:", student.keys())
print("Values:", student.values())
print("Items:", student.items())
Important: Using square brackets to access a non-existent key will raise a KeyError. Use the get() method or check with in operator when you're not sure if a key exists.

Common Dictionary Methods

Python provides many built-in methods for dictionary manipulation. Here are the most useful ones:

Method Description Example
get(key[, default]) Returns value for key, or default if key doesn't exist d.get('x', 0) → value or 0
keys() Returns view of all keys d.keys() → dict_keys(['a','b'])
values() Returns view of all values d.values() → dict_values([1,2])
items() Returns view of all (key, value) pairs d.items() → dict_items([('a',1),('b',2)])
pop(key[, default]) Removes key and returns its value d.pop('a') → 1 (removes 'a')
popitem() Removes and returns last (key, value) pair d.popitem() → ('b', 2)
update([other]) Updates dictionary with key-value pairs from other d.update({'c':3}) → adds 'c':3
clear() Removes all items from dictionary d.clear() → {}
setdefault(key[, default]) Returns value if key exists, else inserts with default d.setdefault('x', 0) → 0 (inserts 'x':0)
# Dictionary methods in action
inventory = {"apples": 50, "bananas": 30, "oranges": 40}

print("Original:", inventory)

# Adding/updating items
inventory["grapes"] = 25 # Add new item
inventory["apples"] = 60 # Update existing item
inventory.update({"pears": 35, "kiwis": 20}) # Add multiple items

print("After updates:", inventory)

# Removing items
removed = inventory.pop("bananas")
print(f"Removed bananas: {removed}")

last_item = inventory.popitem()
print(f"Removed last item: {last_item}")

# Using setdefault
melon_count = inventory.setdefault("melons", 15)
print(f"Melon count: {melon_count}")
print("Inventory:", inventory)

# Iterating through dictionary
print("\nIterating through inventory:")
for item, quantity in inventory.items():
    print(f" {item}: {quantity}")
