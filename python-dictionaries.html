Introduction to Python Dictionaries

In Python, dictionaries are unordered collections of key-value pairs. They are also known as associative arrays or hash maps in other programming languages.

Dictionaries are mutable, unordered, and indexed by keys (which must be immutable types like strings, numbers, or tuples). They are extremely efficient for lookups and are one of Python's most powerful data structures.

Key Characteristics:
  • Unordered collection of key-value pairs
  • Keys must be unique and immutable (strings, numbers, tuples)
  • Values can be of any data type
  • Mutable - can be changed after creation
  • Extremely fast for lookups (O(1) average time complexity)
# Basic dictionary example
person = {
    "name": "Alice",
    "age": 30,
    "city": "New York",
    "is_student": False
}

print(person)
print("Type:", type(person))
print("Length:", len(person))
Output:
{'name': 'Alice', 'age': 30, 'city': 'New York', 'is_student': False}
Type: <class 'dict'>
Length: 4

Creating Dictionaries in Python

Python provides several ways to create dictionaries, each suitable for different scenarios.

Dictionary Creation Methods

Method Example Description
Curly braces {} {'a': 1, 'b': 2} Most common way, with key-value pairs
dict() constructor dict(a=1, b=2) Using keyword arguments (keys as identifiers)
From list of tuples dict([('a',1),('b',2)]) Convert list of (key, value) tuples
Dictionary comprehension {x: x*2 for x in range(5)} Create dynamically from iterable
fromkeys() method dict.fromkeys(['a','b'], 0) All keys with same initial value
# Different ways to create dictionaries
# 1. Using curly braces
dict1 = {"name": "John", "age": 25}

# 2. Using dict() constructor
dict2 = dict(name="Jane", age=30, city="Boston")

# 3. From list of tuples
dict3 = dict([('a', 1), ('b', 2), ('c', 3)])

# 4. Empty dictionary
empty_dict = {}
empty_dict2 = dict()

# 5. Nested dictionary
nested_dict = {
    "user1": {"name": "Alice", "score": 95},
    "user2": {"name": "Bob", "score": 87}
}

print("dict1:", dict1)
print("dict2:", dict2)
print("dict3:", dict3)
print("nested_dict:", nested_dict)

Accessing and Modifying Dictionary Elements

Learn how to retrieve, update, add, and delete elements from dictionaries efficiently.

Accessing Values

# Accessing dictionary elements
student = {
    "name": "John Doe",
    "age": 21,
    "major": "Computer Science",
    "courses": ["Python", "Algorithms", "Database Systems"]
}

# Method 1: Square bracket notation (raises KeyError if key doesn't exist)
name = student["name"]
print("Name:", name)

# Method 2: get() method (safer, returns None or default value)
gpa = student.get("gpa") # Returns None
gpa_safe = student.get("gpa", "Not available") # Returns default
print("GPA:", gpa_safe)

# Method 3: Access nested dictionaries
student["grades"] = {"Python": 95, "Algorithms": 88}
python_grade = student["grades"]["Python"]
print("Python Grade:", python_grade)

Modifying Dictionaries

# Adding and updating dictionary elements
inventory = {
    "apples": 50,
    "bananas": 30,
    "oranges": 25
}

# Adding new key-value pair
inventory["grapes"] = 40

# Updating existing value
inventory["apples"] = 60

# Using update() method for multiple changes
inventory.update({
    "bananas": 35,
    "mangoes": 20,
    "watermelons": 15
})

# Deleting elements
del inventory["oranges"] # Raises KeyError if key doesn't exist

# Safe deletion with pop()
removed_item = inventory.pop("bananas", 0) # Returns 0 if key not found

# Remove last inserted item (Python 3.7+)
last_item = inventory.popitem()

print("Updated Inventory:", inventory)
print("Removed item:", removed_item)
print("Last removed:", last_item)
Best Practice:

Always use the get() method when accessing values if you're unsure whether a key exists. This prevents KeyError exceptions and makes your code more robust.

Essential Dictionary Methods

Python dictionaries come with powerful built-in methods for common operations.

Method Description Example Return Value
keys() Returns a view of all keys dict.keys() dict_keys object
values() Returns a view of all values dict.values() dict_values object
items() Returns a view of (key, value) pairs dict.items() dict_items object
get(key, default) Safely get value with optional default dict.get("x", 0) Value or default
pop(key, default) Remove key and return its value dict.pop("x") Removed value
popitem() Remove and return last (key, value) pair dict.popitem() (key, value) tuple
update() Update with another dict or iterable dict.update(other) None (modifies in-place)
setdefault(key, default) Get value, set to default if key missing dict.setdefault("x", 0) Value
clear() Remove all items dict.clear() None
copy() Return a shallow copy dict.copy() New dictionary
# Practical examples of dictionary methods
employee = {
    "id": 101,
    "name": "Alice Johnson",
    "department": "Engineering",
    "salary": 85000,
    "skills": ["Python", "Django", "AWS"]
}

# 1. keys() - Get all keys
print("Keys:", list(employee.keys()))

# 2. values() - Get all values
print("Values:", list(employee.values()))

# 3. items() - Iterate through key-value pairs
for key, value in employee.items():
    print(f"{key}: {value}")

# 4. setdefault() - Get or set with default
projects = employee.setdefault("projects", [])
projects.append("Inventory System")

# 5. copy() - Create a shallow copy
employee_copy = employee.copy()
employee_copy["name"] = "Bob Smith"

print("\nOriginal:", employee["name"])
print("Copy:", employee_copy["name"])

Dictionary Operations and Iteration

Checking Key Existence

# Different ways to check if a key exists
config = {
    "host": "localhost",
    "port": 8080,
    "debug": True
}

# Method 1: 'in' operator (recommended)
if "host" in config:
    print("Host found:", config["host"])

# Method 2: Using get()
timeout = config.get("timeout")
if timeout is not None:
    print("Timeout:", timeout)

# Method 3: Using keys()
if "port" in config.keys():
    print("Port found:", config["port"])

# Check if value exists
if True in config.values():
    print("Debug mode is enabled")

Dictionary Merging Techniques

# Different ways to merge dictionaries (Python 3.5+)
defaults = {"color": "red", "size": "medium", "price": 100}
user_prefs = {"color": "blue", "discount": 20}

# Method 1: update() (modifies original)
merged1 = defaults.copy()
merged1.update(user_prefs)

# Method 2: Dictionary unpacking (Python 3.5+)
merged2 = {**defaults, **user_prefs}

# Method 3: | operator (Python 3.9+)
merged3 = defaults | user_prefs

# Method 4: Dictionary comprehension
merged4 = {key: user_prefs.get(key, defaults[key]) for key in set(defaults) | set(user_prefs)}

print("Merged (update):", merged1)
print("Merged (unpacking):", merged2)

Dictionary Comprehensions

Dictionary comprehensions provide a concise way to create dictionaries from iterables.

# Basic dictionary comprehension
# Syntax: {key_expression: value_expression for item in iterable}

# 1. Create squares dictionary
squares = {x: x**2 for x in range(1, 6)}
print("Squares:", squares) # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# 2. Transform list to dictionary
fruits = ["apple", "banana", "cherry"]
fruit_lengths = {fruit: len(fruit) for fruit in fruits}
print("Fruit lengths:", fruit_lengths)

# 3. With conditional filtering
numbers = {x: x**3 for x in range(10) if x % 2 == 0}
print("Even cubes:", numbers)

# 4. Transform existing dictionary
prices = {"apple": 1.2, "banana": 0.8, "orange": 1.5}
discounted_prices = {item: price * 0.9 for item, price in prices.items()}
print("Discounted prices:", discounted_prices)

# 5. Switch keys and values (if values are hashable)
original = {"a": 1, "b": 2, "c": 3}
swapped = {value: key for key, value in original.items()}
print("Swapped:", swapped)

# 6. Nested dictionary comprehension
matrix = {
    i: {j: i*j for j in range(1, 4)}
    for i in range(1, 4)
}
print("Multiplication table:", matrix)
Important:

Dictionary comprehensions are great for simple transformations, but for complex logic, consider using regular loops for better readability.

Advanced Dictionary Techniques

DefaultDict from Collections Module

from collections import defaultdict

# 1. Grouping items by category
students = [
    ("Math", "Alice"),
    ("Science", "Bob"),
    ("Math", "Charlie"),
    ("Art", "David"),
    ("Science", "Eve")
]

# Without defaultdict
grouped = {}
for subject, student in students:
    if subject not in grouped:
        grouped[subject] = []
    grouped[subject].append(student)

# With defaultdict (cleaner)
grouped_default = defaultdict(list)
for subject, student in students:
    grouped_default[subject].append(student)

print("Grouped (regular):", grouped)
print("Grouped (defaultdict):", dict(grouped_default))

# 2. Counting with defaultdict
words = ["apple", "banana", "apple", "orange", "banana", "apple"]
word_count = defaultdict(int)
for word in words:
    word_count[word] += 1
print("Word count:", dict(word_count))

OrderedDict for Maintaining Insertion Order

from collections import OrderedDict

# Creating an ordered dictionary
tasks = OrderedDict()
tasks["wake_up"] = "7:00 AM"
tasks["breakfast"] = "8:00 AM"
tasks["work"] = "9:00 AM"
tasks["lunch"] = "1:00 PM"

# Moving an item to the end
tasks.move_to_end("wake_up")

# Inserting at specific position
tasks["exercise"] = "6:00 AM"

print("Ordered tasks:")
for task, time in tasks.items():
    print(f"{task}: {time}")

Practical Real-World Examples

# 1. Configuration Management
class Config:
    def __init__(self):
        self.settings = {
            "database": {
                "host": "localhost",
                "port": 5432,
                "name": "myapp_db"
            },
            "api": {
                "timeout": 30,
                "retries": 3
            }
        }

    def get(self, key, default=None):
        # Nested get using recursion
        keys = key.split(".")
        value = self.settings
        for k in keys:
            value = value.get(k)
            if value is None:
                return default
        return value

config = Config()
print("Database host:", config.get("database.host"))
print("API timeout:", config.get("api.timeout"))

# 2. Cache Implementation
class SimpleCache:
    def __init__(self, max_size=100):
        self.cache = {}
        self.max_size = max_size
        self.access_order = []

    def get(self, key):
        if key in self.cache:
            # Update access order
            self.access_order.remove(key)
            self.access_order.append(key)
            return self.cache[key]
        return None

    def set(self, key, value):
        if len(self.cache) >= self.max_size:
            # Remove least recently used
            oldest = self.access_order.pop(0)
            del self.cache[oldest]

        self.cache[key] = value
        self.access_order.append(key)

    def clear(self):
        self.cache.clear()
        self.access_order.clear()

cache = SimpleCache(max_size=3)
cache.set("user_1", {"name": "Alice"})
cache.set("user_2", {"name": "Bob"})
cache.set("user_3", {"name": "Charlie"})
cache.set("user_4", {"name": "David"}) # Removes user_1

print("Cache contents:", cache.cache)
print("Get user_2:", cache.get("user_2"))

Performance Considerations and Best Practices

Operation Average Time Complexity Worst Case Notes
Get Item O(1) O(n) Extremely fast for lookups
Set Item O(1) O(n) Fast insertion
Delete Item O(1) O(n) Fast deletion
Iteration O(n) O(n) Linear time for all items
Copy O(n) O(n) Deep copy is O(n*m)
Optimization Tips:
  1. Use appropriate key types: Strings and integers make the best dictionary keys due to their fast hash computation.
  2. Avoid using mutable types as keys: Lists and dictionaries cannot be used as keys.
  3. Prefer in operator for membership testing: if key in dict: is faster than if dict.get(key): for checking existence.
  4. Use defaultdict for counting/grouping: Avoids the need for checking key existence manually.
  5. Consider memory usage: Dictionaries have overhead per item (approx. 72 bytes per entry).

Common Pitfalls and Solutions

# Common mistakes and how to avoid them

# 1. Modifying dictionary while iterating
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}

# WRONG: RuntimeError - dictionary changed size during iteration
# for name in scores:
# if scores[name] < 80:
# del scores[name]

# RIGHT: Create list of keys to delete
to_delete = [name for name, score in scores.items() if score < 80]
for name in to_delete:
    del scores[name]

print("Filtered scores:", scores)

# 2. Shallow vs Deep copy
original = {
    "list": [1, 2, 3],
    "nested": {"a": 1, "b": 2}
}

# Shallow copy (references remain)
shallow_copy = original.copy()
shallow_copy["list"].append(4)

# Deep copy (complete copy)
import copy
deep_copy = copy.deepcopy(original)
deep_copy["list"].append(5)

print("Original list:", original["list"]) # [1, 2, 3, 4]
print("Deep copy list:", deep_copy["list"]) # [1, 2, 3, 5]
