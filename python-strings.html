Introduction to Python Strings

In Python, strings are sequences of characters enclosed in quotes. They are one of the most commonly used data types and come with a rich set of built-in methods for manipulation.

Strings in Python are immutable, meaning once created, they cannot be changed. Any operation that appears to modify a string actually creates a new one.

# Basic string examples
print("Hello, World!") # Double quotes
print('Python Programming') # Single quotes
print("""Multi-line strings can be created using triple quotes""") # Triple quotes for multi-line

Creating Strings in Python

Python provides several ways to create strings, each suitable for different scenarios.

String Creation Methods

Method Example Use Case
Single quotes 'Hello' Simple strings without single quotes inside
Double quotes "World" Strings with single quotes inside
Triple quotes '''Multi-line string''' Multi-line strings or docstrings
str() function str(123) Converting other types to strings
# Different ways to create strings
str1 = 'Single quoted string'
str2 = "Double quoted string"
str3 = '''Triple quoted multi-line string'''
str4 = str(42) # Convert integer to string
# String with escape sequences
str5 = "Line 1\nLine 2\tTabbed text"

print(str1)
print(str3)
print(str5)
Output:
Single quoted string
Triple quoted
multi-line string
Line 1
Line 2 Tabbed text

Best Practices:
  1. Use f-strings for modern Python string formatting (Python 3.6+)
  2. Use string methods instead of manual manipulation for better performance
  3. Always sanitize user input strings to prevent security issues
  4. Use triple quotes for docstrings and multi-line strings
  5. Remember that strings are immutable - operations create new strings
