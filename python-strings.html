Python Strings: Complete Guide with Examples

A comprehensive tutorial on Python strings, covering creation, manipulation, methods, formatting, and best practices with practical code examples.

By:Glenn Junsay Pansensoy

Introduction to Python Strings

In Python, strings are sequences of characters enclosed in quotes. They are one of the most commonly used data types and come with a rich set of built-in methods for manipulation.

Strings in Python are immutable, meaning once created, they cannot be changed. Any operation that appears to modify a string actually creates a new one.

# Basic string examples
print("Hello, World!") # Double quotes
print('Python Programming') # Single quotes
print("""Multi-line strings can be created using triple quotes""") # Triple quotes for multi-line

Creating Strings in Python

Python provides several ways to create strings, each suitable for different scenarios.

String Creation Methods

Method Example Use Case
Single quotes 'Hello' Simple strings without single quotes inside
Double quotes "World" Strings with single quotes inside
Triple quotes '''Multi-line string''' Multi-line strings or docstrings
str() function str(123) Converting other types to strings
# Different ways to create strings
str1 = 'Single quoted string'
str2 = "Double quoted string"
str3 = '''Triple quoted multi-line string'''

# String with escape sequences
str5 = "Line 1\nLine 2\tTabbed text"

String Operations

Python supports various operations on strings including concatenation, repetition, indexing, and slicing.

# String concatenation and repetition
str1 = "Hello"
str2 = "World"
result = str1 + " " + str2 # Concatenation
print(result)

repeated = str1 * 3 # Repetition
print(repeated)

# String indexing
text = "Python"
print(text[0]) # First character: 'P'
print(text[-1]) # Last character: 'n'

# String slicing
print(text[0:2]) # Characters from index 0 to 1: 'Py'
print(text[2:]) # From index 2 to end: 'thon'
print(text[::2]) # Every second character: 'Pto'
print(text[::-1]) # Reversed string: 'nohtyP'

Common String Methods

Python provides many built-in methods for string manipulation. Here are some of the most useful ones:

Method Description Example
upper() Converts to uppercase "hello".upper() → "HELLO"
lower() Converts to lowercase "HELLO".lower() → "hello"
strip() Removes whitespace " hello ".strip() → "hello"
split() Splits into list "a,b,c".split(",") → ["a","b","c"]
join() Joins list into string ",".join(["a","b","c"]) → "a,b,c"
replace() Replaces substring "hello".replace("l","x") → "hexxo"
find() Finds substring index "hello".find("e") → 1
# String methods in action
text = " Python Programming "

print("Original:", text)
print("Stripped:", text.strip())
print("Uppercase:", text.upper())
print("Lowercase:", text.lower())
print("Title case:", text.strip().title())

# Split and join
words = text.strip().split(" ")
print("Split:", words)
print("Joined with '-':", "-".join(words))

# Replace and find
print("Replace 'Programming' with 'Coding':", text.replace("Programming", "Coding"))
print("Find 'Pro' index:", text.find("Pro"))

String Formatting

Python offers multiple ways to format strings, including f-strings (recommended), format() method, and % formatting.

# String formatting examples
name = "Glenn"
age = 30
language = "Python"

# 1. f-strings (Python 3.6+) - Recommended
message1 = f"Hello, my name is {name} and I am {age} years old. I love {language}!"
print("f-string:", message1)

# 2. format() method
message2 = "Hello, my name is {} and I am {} years old. I love {}!".format(name, age, language)
print("format():", message2)

# 3. % formatting (older style)
message3 = "Hello, my name is %s and I am %d years old. I love %s!" % (name, age, language)
print("% formatting:", message3)

# Formatting numbers
pi = 3.141592653589793
print(f"Pi rounded to 2 decimals: {pi:.2f}")
print(f"Large number with commas: {1000000:,}")

Advanced String Topics

Escape Sequences

Escape sequences allow you to include special characters in strings:

# Common escape sequences
print("Newline: Line1\nLine2")
print("Tab: Col1\tCol2\tCol3")
print("Backslash: C:\\Users\\Name")
print("Single quote: It\'s raining")
print("Double quote: He said, \"Hello!\"")
print("Raw string (ignores escapes):", r"C:\Users\Name")

String Membership Testing

text = "Python programming is fun"

# Check if substring exists
print("Python" in text) # True
print("Java" not in text) # True
print("python" in text) # False (case-sensitive)
print("python".lower() in text.lower()) # True (case-insensitive)
