Q: What is the difference between is and ==?
== checks for value equality.
is checks for identity (same object in memory).

Common trap: small integers & some strings are interned, so a = 256; b = 256; a is b ‚Üí True, but don't rely on it.
Rule: Use == for values, is for None, True/False, or when you really need identity.
Q: Why are mutable default arguments dangerous? How to fix?
Default arguments are evaluated once at function definition time.
If mutable (list, dict, set), it is shared across calls ‚Üí unexpected side effects. 
def bad(items=[]):
    items.append(1)
    return items

print(bad())  # [1]
print(bad())  # [1,1] ‚Üê surprise!

# Fix: use None + check
def good(items=None):
    if items is None:
        items = []
    items.append(1)
    return items
Q: Explain *args and **kwargs ‚Äì when to use each?
*args ‚Üí collects positional arguments into a tuple.
**kwargs ‚Üí collects keyword arguments into a dict.

Use cases:
  • Forwarding arguments to super() or wrapped functions
  • Flexible APIs (e.g. logging, decorators)
  • Adapter / proxy patterns
def log(msg, *args, level="INFO", **kwargs):
    print(f"[{level}] {msg}", args, kwargs)
Q: What is a closure in Python? Give a practical example.
A closure is a function that remembers variables from its enclosing scope even after that scope is gone.

Classic example ‚Äì counter: 
def make_counter():
    count = 0
    def increment():
        nonlocal count
        count += 1
        return count
    return increment

c1 = make_counter()
print(c1())  # 1
print(c1())  # 2

c2 = make_counter()
print(c2())  # 1  (independent)
Q: @staticmethod vs @classmethod vs regular method ‚Äì when to use?
  • Regular method: takes self, operates on instance
  • @classmethod: takes cls, operates on class (alternative constructors, inheritance friendly)
  • @staticmethod: no self or cls, utility function logically belongs to class namespace
class Date:
    def __init__(self, year, month, day):
        self.year = year

    @classmethod
    def from_string(cls, s):
        y, m, d = map(int, s.split('-'))
        return cls(y, m, d)

    @staticmethod
    def is_leap_year(year):
        return year % 4 == 0 and (year % 100 != 0 or year % 400 == 0)
Q: How does Python's GIL affect threading? When to use threading vs multiprocessing?
GIL (Global Interpreter Lock) ‚Üí only one thread executes Python bytecode at a time.
‚Üí CPU-bound tasks: threading gives no speedup (use multiprocessing)
‚Üí I/O-bound tasks: threading is fine (network, disk, waiting)

Modern alternatives: asyncio (single-threaded concurrency), concurrent.futures, process pools.
Q: Explain context managers and the with statement. How to write your own?
Context managers handle resource acquisition/release (files, locks, db connections).

Two ways:
  1. Class with __enter__ / __exit__
  2. Generator + @contextmanager
from contextlib import contextmanager

@contextmanager
def timer(label):
    import time
    start = time.time()
    try:
        yield
    finally:
        print(f"{label}: {time.time() - start:.3f}s")
Q: What are descriptors? Give a simple example (property-like behavior).
Descriptors are objects that define __get__, __set__, __delete__.
They power @property, class variables with validation, etc. 
class Positive:
    def __set_name__(self, owner, name):
        self.name = name

    def __get__(self, obj, owner=None):
        return obj.__dict__.get(self.name, 0)

    def __set__(self, obj, value):
        if not isinstance(value, (int, float)) or value < 0:
            raise ValueError("Must be non-negative number")
        obj.__dict__[self.name] = value

class Product:
    price = Positive()
