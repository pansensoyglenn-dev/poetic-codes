40 Python String Methods

Number 1: upper()

Transposes all cased characters to their uppercase equivalents.

val = "python".upper() # Output: "PYTHON"
Number 2: lower()

Normalizes all characters to lowercase.

val = "DATA".lower() # Output: "data"
Number 3: casefold()

Implements aggressive case-folding for caseless string comparison across different languages.

val = "Fluß".casefold() # Output: "fluss"
Number 4: capitalize()

Converts the first character to uppercase and all subsequent to lowercase.

val = "hello WORLD".capitalize() # Output: "Hello world"
Number 5: title()

Applies title-casing where every word starts with an uppercase character.

val = "entry point".title() # Output: "Entry Point"
Number 6: swapcase()

Inverts the casing of all characters in the string.

val = "PyThOn".swapcase() # Output: "pYtHoN"
Number 7: find()

Returns the lowest index of a substring; returns -1 on failure.

val = "system".find("y") # Output: 1
Number 8: rfind()

Returns the highest index of a substring; returns -1 on failure.

val = "banana".rfind("a") # Output: 5
Number 9: index()

Identical to find(), but raises a ValueError if the substring is not present.

val = "root".index("o") # Output: 1
Number 10: rindex()

Returns the highest index of a substring; raises ValueError on failure.

val = "root".rindex("o") # Output: 2
Number 11: count()

Returns the number of non-overlapping occurrences of a substring.

val = "tests".count("t") # Output: 2
Number 12: startswith()

Returns True if the string starts with the specified prefix.

val = "v1_b".startswith("v1") # Output: True
Number 13: endswith()

Returns True if the string ends with the specified suffix.

val = "d.csv".endswith(".csv") # Output: True
Number 14: isalnum()

Checks if all characters are alphanumeric (letters or numbers).

val = "user123".isalnum() # Output: True
Number 15: isalpha()

Checks if all characters in the string are alphabetic.

val = "Python".isalpha() # Output: True
Number 16: isascii()

Returns True if the string is empty or all characters are ASCII.

val = "apple".isascii() # Output: True
Number 17: isdecimal()

Checks if all characters are decimals (0-9).

val = "100".isdecimal() # Output: True
Number 18: isdigit()

Checks if all characters are digits (includes superscripts).

val = "²".isdigit() # Output: True
Number 19: isnumeric()

Checks if all characters are numeric (includes fractions).

val = "½".isnumeric() # Output: True
Number 20: isidentifier()

Determines if the string is a valid identifier (variable name) in Python.

val = "my_var".isidentifier() # Output: True
Number 21: islower()

Checks if all cased characters are lowercase.

val = "low".islower() # Output: True
Number 22: isupper()

Checks if all cased characters are uppercase.

val = "HIGH".isupper() # Output: True
Number 23: istitle()

Returns True if the string is in title-case.

val = "Hello World".istitle() # Output: True
Number 24: isprintable()

Checks if characters are printable (e.g., no escape codes).

val = "data".isprintable() # Output: True
Number 25: isspace()

Checks if the string contains only whitespace.

val = " \t ".isspace() # Output: True
Number 26: center()

Centers the string in a field of specific width using a padding character.

val = "X".center(5, "-") # Output: "--X--"
Number 27: ljust()

Left-justifies the string with padding.

val = "txt".ljust(5, "*") # Output: "txt**"
Number 28: rjust()

Right-justifies the string with padding.

val = "txt".rjust(5, "*") # Output: "**txt"
Number 29: zfill()

Pads a numeric string with leading zeros.

val = "42".zfill(5) # Output: "00042"
Number 30: expandtabs()

Sets the tab size of the string to the specified number of spaces.

val = "A\tB".expandtabs(2) # Output: "A B"
Number 31: split()

Splits string into a list based on a separator.

val = "a,b,c".split(",") # Output: ["a", "b", "c"]
Number 32: rsplit()

Splits string into a list starting from the right side.

val = "a,b,c".rsplit(",", 1) # Output: ["a,b", "c"]
Number 33: splitlines()

Splits the string at line breaks into a list.

val = "l1\nl2".splitlines() # Output: ["l1", "l2"]
Number 34: join()

Concatenates an iterable's elements into a single string using a joiner.

val = "-".join(["a", "b"]) # Output: "a-b"
Number 35: partition()

Splits the string into a 3-tuple (before, sep, after) at first match.

val = "a-b-c".partition("-") # Output: ("a", "-", "b-c")
Number 36: rpartition()

Splits the string into a 3-tuple at the last match.

val = "a-b-c".rpartition("-") # Output: ("a-b", "-", "c")
Number 37: strip()

Removes leading and trailing whitespace/chars.

val = " x ".strip() # Output: "x"
Number 38: lstrip()

Removes leading whitespace/chars.

val = " x ".lstrip() # Output: "x "
Number 39: rstrip()

Removes trailing whitespace/chars.

val = " x ".rstrip() # Output: " x"
Number 40: replace()

Substitutes all occurrences of a substring with another.

val = "old".replace("old", "new") # Output: "new"
