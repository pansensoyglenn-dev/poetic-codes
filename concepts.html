Dynamic Typing

Python is strongly, dynamically typed. Objects have types, but variables are just names pointing to those objects.

x = 100  # x points to an int
x = "Now a string"  # x now points to a str

First-Class Functions

Functions can be passed as arguments, returned from other functions, and assigned to variables.

def greet(name): return f"Hello, {name}"

def apply_func(f, val): return f(val)

print(apply_func(greet, "Dev"))

Python Decorators

Decorators allow you to "wrap" another function to extend its behavior without permanently modifying it. This is a common pattern for logging, authentication, and timing.

def log_call(func):
    def wrapper(*args, **kwargs):
        print(f"Calling {func.__name__}")
        return func(*args, **kwargs)
    return wrapper

@log_call
def fetch_data():
    return {"data": "secure"}

Exception Lifecycle

Handle runtime errors gracefully. The finally block ensures resources (like file handles) are released.

try:
    perform_action()
except Exception as e:
    print(f"Error: {e}")
finally:
    cleanup()
