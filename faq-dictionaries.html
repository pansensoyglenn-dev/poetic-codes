Python Dictionaries FAQ

A dictionary is a built-in data type that stores data in key-value pairs. They are ordered (as of Python 3.7+), mutable, and do not allow duplicate keys.
You can use curly braces my_dict = {} or the constructor my_dict = dict().
Use square brackets: my_dict["key"]. Alternatively, use my_dict.get("key") to avoid a KeyError if the key doesn't exist.

Keys must be immutable (hashable) types, such as strings, numbers, or tuples. Lists and other dictionaries cannot be keys.
Simply assign a value to a key: my_dict["new_key"] = "new_value". You can also use my_dict.update({"key": "value"}).
Use del my_dict["key"] or my_dict.pop("key"). The latter returns the value of the removed item.

Use for key, val in my_dict.items(): to get both, or .keys() and .values() for specific components.
